## Development

### Installing Dependencies

From within the root directory:

with npm

```sh
npm install
```

with yarn

```sh
yarn install
```

### Tech Stack

#### Front-end

- ReactJs
- React Routers
  - Dynamic URL routing is handled using React Routers

#### Back-end

- Node
  - for serving reactJs build app
